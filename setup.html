📋 Setup Guide — Hardware & Software Stack

Complete instructions for setting up your Home Solar IoT system.

🔍 Step 1: Discover Your Inverter

🔎 Identify Your Inverter Brand

1
Find the model sticker on the side of your inverter. Common WiFi brands: Growatt, Deye, Sofar, SolarEdge, Goodwe, Solis, Voltronic
2
Check if there's a WiFi stick/dongle plugged into the inverter
3
Check your router for the inverter's IP address

📡 Scan For Open Ports (on Pi4)

# Find inverter on network
nmap -sn 192.168.1.0/24

# Scan inverter ports
nmap -p 502,8899,8888,80 192.168.1.XXX
Port 502 = Modbus TCP
Port 8899 = SolarmanV5
Port 80 = Web UI

🛠️ Step 2: Hardware Wiring

📶 Option A — WiFi Only (Easiest)

1
Pi4 connects to home WiFi → polls inverter WiFi logger → reads data via SolarmanV5 or Modbus TCP
2
ESP32 connects to home WiFi → posts PZEM-004T load readings to Pi4 via MQTT
pip3 install pysolarmanv5
python3 -c "from pysolarmanv5 import PySolarmanV5"

🔧 Option B — ESP32 + RS485

⚠ Swap A/B if no data!

⚡ PZEM-004T Load Meter

1
Clamp CT coil around main live wire at distribution board
2
Connect PZEM to ESP32 via TX/RX pins
PZEM TX → GPIO16
PZEM RX → GPIO17
5V → VIN, GND → GND
💻 Step 3: Software Stack

🐳 Install Docker on Pi4

# Install Docker
curl -fsSL https://get.docker.com | sh
sudo usermod -aG docker pi

# Run services
docker run -d mosquitto
docker run -d influxdb
docker run -d grafana/grafana

🔄 Data Pipeline Flow

1
Python script polls inverter → publishes to MQTT solar/generation
2
ESP32 reads PZEM → publishes to MQTT solar/load
3
Node-RED subscribes → writes to InfluxDB → sends alerts

🧹 Panel Cleaning Algorithm

1
Call OpenWeatherMap API for irradiance data
2
Calculate expected = 3500W × irradiance
3
If actual/expected < 0.80 → flag for cleaning
efficiency = actual / expected
if efficiency < 0.80:
  send_telegram_alert()
